Style (micro) guide
- Use 4 spaces for indentation (no tabs).
- CapWords for class names.
- lowercase_with_underscores for functions and variables. 

                >>> import this # Try this
Variables
- In Python every variable is an object.
- You do not need to declare variables before using them. 

                >>> my_bool = True # Bool
                
>>> my_int = 7 # Int
                
>>> example_float = 7.0 # Float
                
>>> my_name = 'Matt' # String
Strings
- Python strings are arrays of bytes representing unicode characters.
- Single character in a string is a string with a length 1. 

                >>> name = "Cartman" # New string
                
>>> name[4] # Get 5th character (index starts at 0)
                
'm'
                
>>> name[-1] # Counting backwards starts from -1
                
'n'
                
>>> name[2:5] # Get characters with index in interval [2:5)
                
'rtm'
                
>>> name + ' is super cool' # Combine strings
                
'Cartman is super cool'
                
>>> 'rt' in name # Check if string occurs in other string
                
True
Tuples
- Tuple in Python is a equence of immutable objects.
- Tuple cannot be changed. 

                >>> >>> tuple = ('One', 'Two', 3, 4) # New tuple
                
>>> tuple[1] # Get item at index 1 (index starts at 0)
                
'Two'
                
>>> tuple[1] = 'Three' # Cannot change it
                
>>> # Should get TypeError: 'tuple' object does not support item assignment
                
>>> len(tuple)
                
4
Lists
- List in Python is mutable, changeable and ordered sequence of elements.
- It is a collections of items with assigned index to each item. 

                >>> names = ['Stan', 'Kyle', 'Eric', 'Kenny', 'Butters', 'Jimmy'] # New list
                
>>> names[1] # Get 5th character (index starts at 0)
                
'Kyle'
                
>>> names[-1] # Similar to strings behaviour
                
'Jimmy'
                
>>> names[2:4] # List slicing
                
['Eric', 'Kenny']
                
>>> len(names) # Number of items
                
6
                
>>> names + ['Chef', 'The Coon', 'Professor Chaos'] # Combine lists
                
['Stan', 'Kyle', 'Eric', 'Kenny', 'Chef', 'The Coon', 'Professor Chaos']
                
>>> 'Kyle' in names # Check if string occurs in a list
                
True
Dictionaries
- Stores key-value pairs.
- Unordered, changeable and indexed. 

                >>> mustang_gt = {'manufacturer' : 'Ford',
                
... 'power' : '460bhp',
                
... 'torque' : '570Nm'}
                
>>> mustang_gt['torque'] # access dictionary value
                
'570Nm'
                
>>> mustang_gt['compression_ratio'] = '12.0:1' # set new key-value
IF statements
- Evaluates if statement is true or false.
- Runs code only if condition is true. 

                >>> number_five = 5
                
>>> number_ten = 10
                
>>> number_one = 1
                
>> if number_five > number_ten:
                
...     print('5 > 10')
                
... elif number_five == number_one:
                
...     print('5 == 1')
                
... elif number_five == 5:
                
...     print('5 == 5')
                
... else:
                
...     print('Sorry..')
                
5 == 5
                
>>>
                
>>># You can use and or not
                
>>> if 'Eric' in names and 'Butters' in names or not False:
                
...     print(True)
                
...
                
True
FOR loop
- Python FOR statement iterates over a sequence in order. - Executes a sequence of statements multiple times.
- Break statement stops the loop before it has looped through all the items 

                >>> games = ['Wolfenstein', 'Doom', 'Mortal Kombat', 'Dead Space']
                
>> for game in games:
                
...     print(game)
                
...     if game == 'Mortal Kombat':
                
...         break
                
Wolfenstein
                
Doom
                
Mortal Kombat
